fromEvent()


Lista de eventos

Parametros:


1. target (requerido) El elemento DOM, EventEmitter, objeto jQuery u otro objeto que soporte la adición/eliminación de event listeners. Puede ser:
Elementos DOM: document, window, un elemento HTML específico Node.js EventEmitter Objetos jQuery Cualquier objeto con métodos addEventListener/removeEventListener o on/off


2. eventName (requerido)
El nombre del evento como string: 'click', 'mousemove', 'keyup', etc.


3. options (opcional) Un objeto con opciones para el event listener. Las opciones comunes incluyen:
  • capture: Boolean para usar capture phase
  • passive: Boolean para indicar que el listener nunca llamará a preventDefault()
  • once: Boolean para que el listener se ejecute solo una vez
